# 資産価値査定アプリ（時計特化）

品名・型番を入力すると、**価格.com と 楽天市場の販売価格**を集計して時計の現在の市場価値を査定する Web アプリです。検索後は**型番で絞り込み**でき、査定した品物を**マイ・ポートフォリオ**に登録してブランド別に資産を管理できます。APIキー不要で、Netlify にデプロイするだけで動作します。

## ⚠️ 重要な注意事項

価格.com・楽天市場とも公開 API を使わず、本アプリは**検索結果ページの HTML を取得・解析（スクレイピング）**しています。次のリスクを理解した上でご利用ください。

- 各サイトの利用規約に抵触する可能性があります。**個人利用・社内検証の範囲に留め、一般公開・商用提供は推奨しません。**
- ページ構造が変更されると動作しなくなります（`netlify/functions/search.js` を修正してください）。特に楽天はReact製でクラス名がハッシュ化されるため、`item.rakuten.co.jp` リンクと `data-price` 属性に依存しています。
- アクセスが集中すると参照元にブロックされる可能性があります（レスポンスは5分間キャッシュしています）。

## 機能

### 1. 市場価値査定
品名・型番で検索すると、価格.com・楽天市場の実売価格から以下を算出します。

- **推定市場価値** … 販売価格の中央値（外れ値除外後）
- **価格レンジ** … 最安値／中央値／平均値／最高値／参照件数

### 2. 型番で絞り込み（ドロップダウン）
検索結果の商品名から型番（リファレンス番号）を自動抽出し、ドロップダウンを表示します。

- 選択肢は **「親キーワード ＋ 型番」** で表示・検索します（型番のみだと無関係な商品が混ざるため、必ず親キーワードとセットで絞り込みます）。
- ブランドごとに異なる型番形式に対応：ロレックス `126610LN` ／ オメガのドット型番 `310.30.42.50.01.002` ／ グランドセイコー `SBGA465` ／ G-SHOCK `GMW-B5000D-1JF` ／ IWC `IW3777` など。
- ケース径（42MM）・年号（2025）・ショップ独自SKU は型番候補から除外します。

### 3. マイ・ポートフォリオ（ブランド別資産管理）
査定結果を「ポートフォリオに追加」すると以下が見られます。

- **総資産** … 保有品の推定市場価値の合計
- **ブランド別内訳** … メーカー名で自動集計（バーグラフ・構成比%）
- **保有一覧** … 品名・ブランド・推定市場価値・参照元・価格取得日
- **価格を更新** … 全保有品の価格を再取得して洗い替え

保存先はブラウザの localStorage です（サーバーには保存されません）。

### 査定精度を上げる仕組み

1. 商品名のみ検索（価格.com `nmonly=1`）でアクセサリーの混入を抑制
2. カテゴリ絞り込み … 最頻カテゴリと同カテゴリのみ採用（カテゴリ不明の楽天はトークン一致で判定）
3. 検索語の全単語一致 … 文字種境界を考慮した照合で別モデルを除外（「Mark II」に「Mark III」が混ざらない）
4. 外れ値除外 … 四分位範囲（IQR）の1.5倍を超える価格を統計から除外

### 動作確認済みの時計ブランド

ロレックス（サブマリーナ・デイトジャスト）／オメガ（スピードマスター・シーマスター）／チューダー／IWC／カルティエ／ブライトリング／パテックフィリップ／タグホイヤー／カシオ（G-SHOCK）／シチズン など。

## デプロイ（Netlify）

1. このフォルダを GitHub リポジトリに push（または Netlify CLI で直接デプロイ）
2. Netlify で新規サイトを作成しリポジトリを接続（ビルド設定は `netlify.toml` が自動適用）
3. 環境変数の設定は不要。デプロイ完了後そのまま利用できます。

## ローカルで動かす

```bash
node dev-server.js
# → http://localhost:8888
```

## ファイル構成

```
├── netlify.toml                 # Netlify 設定（公開dir / functions / リダイレクト）
├── package.json
├── dev-server.js                # ローカル確認用の簡易サーバー
├── netlify/
│   └── functions/
│       ├── search.js            # 査定API（価格.com・楽天市場の解析／型番抽出／ブランド判定）
│       └── lib/common.js        # 共有ユーティリティ（取得・統計・整形）
└── public/
    ├── index.html               # 画面
    ├── css/style.css            # Chrono24風ダークUI
    └── js/app.js                # 査定・型番絞り込み・ポートフォリオロジック
```
