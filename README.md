# 資産価値査定アプリ（時計特化）

品名・型番を入力すると、**価格.com・楽天市場・Yahoo!ショッピングの販売価格**を集計して時計の現在の市場価値を査定する Web アプリです。検索後は**型番で絞り込み**でき、査定した品物を**マイ・ポートフォリオ**に登録してブランド別に資産を管理できます。APIキー不要で、Netlify にデプロイするだけで動作します。

## ⚠️ 重要な注意事項

各サイトとも公開 API を使わず、本アプリは**検索結果ページの HTML を取得・解析（スクレイピング）**しています。次のリスクを理解した上でご利用ください。

- 各サイトの利用規約に抵触する可能性があります。**個人利用・社内検証の範囲に留め、一般公開・商用提供は推奨しません。**
- ページ構造が変更されると動作しなくなります（`netlify/functions/search.js` を修正してください）。各サイトの解析方法は下記の通りです。
  - 価格.com … 検索結果ページのHTML（Shift_JIS）
  - 楽天市場 … React製のためクラス名はハッシュ化。`item.rakuten.co.jp` リンクと `data-price` 属性に依存
  - Yahoo!ショッピング … Next.js製。`__NEXT_DATA__` のJSONから商品データを抽出
- アクセスが集中すると参照元にブロックされる可能性があります（レスポンスは5分間キャッシュしています）。

調査したが採用できなかったサイト：Chrono24（bot遮断・403）、GINZA RASIN／コメ兵（検索結果がJavaScript描画でサーバーから取得不可）、買取系サイト各種（JS描画・bot遮断・応答遅延）。

## 機能

### 1. 市場価値査定
品名・型番で検索すると、価格.com・楽天市場・Yahoo!ショッピングの実売価格から以下を算出します。

- **推定市場価値** … 販売価格の中央値（外れ値除外後）
- **価格レンジ** … 最安値／中央値／平均値／最高値／参照件数

### 2. 段階選択（ブランド → モデル → 型番 → 文字盤）
機械式時計のブランドを網羅し、上から順に絞り込めます。

- **① ブランド** … ロレックス〜独立系まで約50ブランド（機械式のみ。G-SHOCK等のクオーツ／デジタルは対象外）
- **② モデル** … 選んだブランドの系列（例：ロレックス→サブマリーナ・デイトナ…）。必ず「ブランド＋モデル」で検索するため、デイトナ等のモデル名で他ブランドが混ざらない
- **③ 型番** … 検索結果から自動抽出し、**現行モデルから順**に表示（型番先頭の数値の降順。例：126500→116520→16520）
- **④ 文字盤** … 文字盤の色（黒／白／メテオライト等）で絞り込み。同じ型番でも色で相場が変わる（例：デイトナ 白¥5.59M／黒¥5.30M）

型番はブランドごとに異なる形式に対応：ロレックス `126500LN` ／ オメガのドット型番 `310.30.42.50.01.002` ／ グランドセイコー `SBGA465` など。下部の検索ボックスで型番直打ちも可能です。

### 3. マイ・ポートフォリオ（ブランド別資産管理）
査定結果を「ポートフォリオに追加」すると以下が見られます。

- **総資産** … 保有品の推定市場価値の合計（売却済みは除外）
- **ブランド別内訳** … ブランド名で自動集計（バーグラフ・構成比%）。表記ゆれ（ROLEX／ロレックス）やモデル名（デイトナ→ロレックス）は正規化して統一
- **写真** … 各銘柄に自分で撮った写真を登録可（端末内で縮小して localStorage に保存）。未登録時は検索結果の商品画像を初期表示
- **売却済みの記録** … 手放した時計は「売却」ボタンで売却済みに（任意で売却価格も記録）。総資産・内訳からは除外し、売却額合計を集計
- **価格を更新** … 保有中の全銘柄の価格を再取得して洗い替え

保存先はブラウザの localStorage です（サーバーには保存されません）。

### 4. TOPページの注目モデル
デイトナ（白）・ノーチラス・ロイヤルオークの3本を写真付きで表示（実売画像と中央値）。クリックでそのまま査定します。

### 査定精度を上げる仕組み（時計本体以外を除外）

1. 商品名のみ検索（価格.com `nmonly=1`）でアクセサリーの混入を抑制
2. カテゴリ絞り込み … 最頻カテゴリと同カテゴリのみ採用（カテゴリ不明の楽天／Yahoo はトークン一致で判定）
3. 検索語の全単語一致 … 文字種境界を考慮した照合で別モデルを除外（「Mark II」に「Mark III」が混ざらない）
4. **付属品・非本体の除外** … 商品名から「ラバーベルト／社外品／ワインダー／保護フィルム／オーバーホール（修理）／攻略本・ムック」等を判定して除外（革ベルトの本体を巻き込まないよう語は限定）
5. **価格外れ値の除外** … IQR上限で高値の無関係品を除外。さらに中央値の12%未満は本体ではない（本・修理サービス・他ブランドのキーワードスパム等）とみなして除外

### 対応ブランド（機械式・約50ブランド）

※ udedokeitoushi.com を参考に機械式ブランドを網羅。G-SHOCK・カシオ等のクオーツ／デジタルは対象外。香水・バッグが多く相場が乱れるブランド（モンブラン等）は除外しています。

## デプロイ（Netlify）

1. このフォルダを GitHub リポジトリに push（または Netlify CLI で直接デプロイ）
2. Netlify で新規サイトを作成しリポジトリを接続（ビルド設定は `netlify.toml` が自動適用）
3. 環境変数の設定は不要。デプロイ完了後そのまま利用できます。

## ローカルで動かす

```bash
node dev-server.js
# → http://localhost:8888
```

## ファイル構成

```
├── netlify.toml                 # Netlify 設定（公開dir / functions / リダイレクト）
├── package.json
├── dev-server.js                # ローカル確認用の簡易サーバー
├── netlify/
│   └── functions/
│       ├── search.js            # 査定API（価格.com・楽天市場の解析／型番抽出／ブランド判定）
│       └── lib/common.js        # 共有ユーティリティ（取得・統計・整形）
└── public/
    ├── index.html               # 画面
    ├── css/style.css            # Chrono24風ダークUI
    └── js/app.js                # 査定・型番絞り込み・ポートフォリオロジック
```
